The user study instructions is included here in the 'instructions.pdf' file.

For convenience, screenshots are also included for examples of each type of question user may get.

To acquire all original materials in whatever format you desire, go to the 'Sample Materials' folder, add the ImageNet contributions data acquired from Experiment 2, place the ImageNet dataset in a 'data' folder, and open the jupyter notebook.

All details are in it, with a simple pipeline to sample new materials, and the original materials, should you wish to do a similar study.

You may also alter the alpha hyperparameter if desired to see the effect on explanations.
